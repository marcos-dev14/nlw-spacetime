# Rocketseat NLW Spacetime (Node JS, Next JS, Expo)

Evento da [Rocketseat](https://www.youtube.com/c/RocketSeat). Nesse evento criamos o projeto Cápsula do tempo, esse projeto completo com Node JS, React (Next) e Mobile (Expo). Nela podemos adicionar fotos suas com a data atual para salvar alguns momentos que você queira. Esse projeto foi um projeto feito pela equipe da Rocketseat na NLW Spacetime, uma semana de muito conteúdo e muito estudos...
